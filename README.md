
# Artificial Intelligent System - Intelligent Models (Collezione Esami)

Questo repository contiene una collezione di tre esami per il corso **"Artificial Intelligent System - Intelligent Models"**. I materiali sono organizzati in tre cartelle principali, ognuna corrispondente a un diverso esame e ai relativi argomenti.

## 📂 Struttura del Repository

Il progetto è organizzato come segue:

| Cartella | Numero Esame | Descrizione / Argomento |
| :--- | :--- | :--- |
| `Ricerca_Spazio_stati` | **Esame 1** | Focalizzato su algoritmi di ricerca in spazi di stati e logica. |
| `PDDL` | **Esame 2** | Focalizzato sul Planning Domain Definition Language (PDDL) e pianificazione automatica. |
| `facebook_data_analisys` | **Esame 3** | Copre le tecniche di analisi dei dati applicate a dataset di social media (Facebook). |

## 📋 Panoramica dei Contenuti

### 1. Ricerca Spazio Stati
Questa sezione contiene i materiali del primo esame. Si concentra sugli algoritmi per navigare in spazi di stati, inclusi euristiche di ricerca e logica di individuazione del percorso (*pathfinding*).

### 2. PDDL
Il secondo esame si focalizza sul **Planning Domain Definition Language**. Questo include la definizione di stati iniziali, obiettivi, azioni e precondizioni per risolvere problemi complessi di pianificazione in sistemi di IA.

### 3. Analisi Dati Facebook
L'ultima sezione contiene i materiali del terzo esame, incentrati sulle tecniche di analisi dei dati, potenzialmente coinvolgendo machine learning o elaborazione statistica di dataset relativi ad ambienti di social media.

## 🚀 Come utilizzare questo repository
Questo repository è destinato a studenti e ricercatori per studiare il percorso del corso "Intelligent Models". È possibile navigare tra le cartelle per trovare problemi specifici, script o documentazione rilevante per ogni fase dell'esame.
